# Gran Logia Simbólica Mixta del Paraguay (GLSMRNP)

Página web oficial de la **Gran Logia Simbólica Mixta del Paraguay**. Desarrollada con estándares premium de performance, diseño oscuro y animaciones sutiles, optimizada especialmente para despliegue automatizado.

## 🌟 Características

* **Diseño Ultra-Premium:** Estética esotérica elegante ("Dark Matter"), diseño *Glassmorphism* y componentes con levitación asincrónica.
* **Rendimiento "Zero-Config":** Archivo monolítico hiper optimizado garantizando carga milimétrica en servidores estáticos sin intermediarios.
* **Multilenguaje Dinámico:** Integración nativa asincrónica en Español, Guaraní, Jopará, Inglés y Portugués.
* **Mapa Estático Inmersivo:** Localización del mapa vía Leaflet sin interferir en la fricción de navegación del usuario móvil.
* **Caché Ultrasónica (Vercel):** Reglas específicas para almacenar en memoria caché del visitante los componentes más pesados (Carta Patente en PDF) para un rendimiento brutal en re-vistas.

## 🚀 Despliegue Continuo (Vercel)

El proyecto está estructurado con el archivo `vercel.json` para ser reconocido automáticamente. No requiere paso de compilación (`build`).

1. Sincroniza este repositorio en tu cuenta de GitHub.
2. Impórtalo desde el dashboard de Vercel.
3. Despliegue en segundos.
